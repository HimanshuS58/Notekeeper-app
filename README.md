# Notekeeper APP
This project is a full-stack note-taking application built with React, Express.js, and MongoDB, allowing users to create, view, update, and delete notes with persistent data storage. The frontend uses React, Context API, React Router, and Tailwind CSS for responsive UI and smooth navigation, while the backend exposes RESTful APIs using Express and Mongoose.
To run this project locally, you must configure environment variables for the backend (MongoDB connection string and server port) and install dependencies for both the frontend and backend before starting the application.
